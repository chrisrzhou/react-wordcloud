## Getting Started

Simply provide an array of words with valid `text` and `count` fields to begin!

<Playground>
  <ReactWordcloud words={words} />
</Playground>

## Setting Wordcloud Size

By default, `ReactWordcloud` inherits the size of the parent container responsively. You can override this by providing explicit width and height values in the `size` prop.

### Responsive parent container

<Playground>
  <ReactWordcloud words={words} />
</Playground>

### Explicit size values

<Playground>
  <ReactWordcloud
    words={words}
    options={{ ...defaultOptions, size: [600, 600] }}
  />
</Playground>

## Update Transitions

`ReactWordcloud` handles updating D3 transitions when `words`, `size` and `options` prop change. The following example demonstrates this behavior when their respective buttons are clicked.

<Playground>
  {() => {
    const [updatingOptions, setUpdatingOptions] = useState(defaultOptions);
    const [updatingWords, setUpdatingWords] = useState(words);
    const [size, setSize] = useState([400, 300]);
    return (
      <>
        <button onClick={() => setUpdatingOptions({ ...defaultOptions })}>
          Update options
        </button>
        <button
          onClick={() => {
            setUpdatingWords([...words].slice(0, choose([50, 100])));
          }}
        >
          Update words
        </button>
        <button
          onClick={() => {
            setSize(choose([[600, 300], [400, 300]]));
          }}
        >
          Update size
        </button>
        <ReactWordcloud
          words={updatingWords}
          size={size}
          options={updatingOptions}
        />
      </>
    );
  }}
</Playground>

## Callbacks

`ReactWordcloud` provides a `callbacks` prop to configure detailed word colors, tooltips and events.

### Word Colors and Tooltips

`ReactWordcloud` applies a random color palette to word colors. It also defaults to selecting the `text` field for rendering tooltips. In the example below, we can pass custom `getWordColor` and `getWordTooltip` callbacks to update word colors and tooltips in data-driven way.

<Playground>
  <ReactWordcloud
    words={words}
    callbacks={{
      getWordColor: word => {
        return "red";
      },
      getWordTooltip: word => {
        return "adfadfdf";
      }
    }}
  />
</Playground>

### Word Events

You can pass callbacks such as `onWordClick`, `onWordMouseEnter` and `onWordMouseLeave` to capture various events.

<Playground>
  {() => {
    const [callbackLog, setCallbackLog] = useState({callback: '', log: ''});

    function getCallback(callback) {
      return function(word) {
        setCallbackLog({
          callback,
          log: JSON.stringify(word, null, 2),
        });
      }
    }

    return (
      <>
        <ReactWordcloud
          words={words}
          callbacks={{
            onWordClick: getCallback('onWordClick'),
            onWordMouseOut: getCallback('onWordMouseOut'),
            onWordMouseOver: getCallback('onWordMouseOver'),
          }}
        />
        <h3>Callback Log: {callbackLog.callback}</h3>
        <pre style={{fontSize: 12}}>
          {callbackLog.log}
        </pre>
      </>
    );

}}

</Playground>
